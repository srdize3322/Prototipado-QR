# Wi-Fi - Conectividad

## Estado: ✅ Integrado

**Todas las plataformas propuestas incluyen Wi-Fi integrado.**  
No se requiere módulo externo.

---

## Comparativa

| Plataforma | Bandas | Alcance |
|------------|--------|---------|
| **ESP32-DevKit** | 2.4GHz | 50-100m |
| **ESP32-S3** | 2.4GHz | 50-100m |
| **Orange Pi** | 2.4/5GHz | 30-80m |
| **RPi Zero 2W** | 2.4/5GHz | 30-80m |
| **Arduino MKR** | 2.4/5GHz | 50-100m |

---

## Consideraciones

**Alcance:**  
- Abierto: 50-100m  
- Interior: 20-40m  
- Mejora: Antena externa (+$2)

**Consumo:**  
- ESP32: 160mA activo, 20mA sleep  
- RPi/OPi: 300-500mA constante

**Seguridad:**  
✅ WPA2/WPA3 soportado  
✅ TLS para HTTPS  
❌ Evitar WEP/Open
