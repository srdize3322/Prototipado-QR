# Ejemplos de Código

Este directorio contiene ejemplos de código de referencia para implementar el módulo lector QR.

## Estructura

- `esp32/`: Ejemplos para ESP32
- `raspberry-pi/`: Ejemplos para Raspberry Pi
- `backend/`: Ejemplos de código backend
- `schemas/`: Esquemas de mensajes y datos

## Nota

Estos son ejemplos educativos y de referencia. Para implementaciones de producción, consultar los repositorios específicos de firmware y backend.
